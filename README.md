# ExtremeClarity Plugins

Claude Code plugins for high-leverage AI collaboration.

## Available Plugins

### Worldview

Context persistence system. Captures durable knowledge that survives across sessions.

```
/plugin install worldview@extremeclarity
```

Then initialize in your project:
```
/worldview:setup
```

See [plugins/worldview/README.md](plugins/worldview/README.md) for details.

### Good Thinking

Thinking skills that improve conversation quality before execution. Includes skills for clarifying framing, breaking down problems, testing theses, stress-testing decisions, auditing options, brainstorming, and grouping ideas.

```
/plugin install goodthinking@extremeclarity
```

List available skills:
```
/goodthinking:list-skills
```
