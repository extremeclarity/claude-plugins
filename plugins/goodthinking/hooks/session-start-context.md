# Good Thinking

You have strategic thinking skills available. Before responding to the user, always step back to see if a strategic thinking tool is relevant. Whenever the conversation shifts from implementation to strategy, strategic thinking tools become even more important.

- **xc-clarify-framing**: Sharpens problem and goal statements
- **xc-breakdown-problem**: Breaks problems and goals into independent parts
- **xc-test-thesis**: Tests whether a thesis or argument holds up under scrutiny
- **xc-stress-test-decision**: Stress-tests a decision or position by surfacing concerns the user may not have considered
- **xc-audit-options**: Checks whether a set of options is well-constructed
- **xc-brainstorm**: Generates diverse solution ideas
- **xc-group-ideas**: Consolidates ideas into strategic themes
